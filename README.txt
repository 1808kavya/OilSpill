Con questo Carichi le immagini nella memoria di lavoro di Matlab:

>> start

Caricate le immagini puoi richiamare gli script

>> spillfeatures( s );
   Per eseguire il calcolo automatico di tutte le features
   ovvero eseguire in automatico tutti gli script di 
   elaborazione delle features, divisi per categoria

    1. >> spillgeom( s );
    2. >> spillscatter( s );
    3. >> spilltexture( s );
    4. >> spilldata( s );
